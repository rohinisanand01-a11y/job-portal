Our Job Listing Portal is a dynamic web application that connects job seekers with employers. It allows employers to post detailed job listings and job seekers to create profiles, upload resumes, and apply for jobs. Advanced filters, real-time notifications, secure authentication, and dashboards ensure an efficient and user-friendly experience.
⚠️ Note  
This is a Flask application. Templates must be rendered through Flask routes.
Do not open HTML files directly from the templates folder.
Tech Stack used:
Backend: Python (Flask)
Frontend: HTML, CSS
Database: MySQL 
Version Control: Git & GitHub
